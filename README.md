# gourd-cmd
了解下mmp技术，练手项目

*****启动*****
 - 注意事项
```
1. -o 为操作类型 目前只支持 verify
2. -p 为需要记住的私钥即密码, 为了保证安全性，必须为16位字符
3. 所有账户信息都是根据私钥进行加密存储在文件同目录的/gourd/gourd文件中
```

 - 打开终端，进入存放启动文件的目录
 
windows
```
start ./gourd.exe -o verify -p 1234567890123456
```
linux或mac
```
./gourd -o verify -p 1234567890123456
```
